创建自己的博客：（一）搭建hexo博客

本文所有的软件和配置是在Windows下进行的

安装Git

　　Git的客户端很多，本文使用的是msysgit，下载下来并按步骤执行安装即可。 　　

安装Node

　　在Windows环境下安装Node.js非常简单，仅须到Node.js官网下载相应平台的最新版本，一路安装即可。

安装hexo

　　Node和Git安装完成之后，利用npm命令即可安装hexo。（在任意位置点击鼠标右键，选择Git bash）

npm install -g hexo

创建hexo文件夹

　　安装完成后，在指定的文件夹下（如Ｄ:\hexo），执行init指令(在Ｄ:\hexo内点击鼠标右键，选择Git bash)，Hexo即会自动在该文件夹下建立网站所需要的所有文件。

hexo init

本地查看

　　现在已经搭建起一个基本的本地hexo博客了，执行以下命令(在Ｄ:\hexo)，然后到浏览器输入localhost:4000看看。

hexo generate
hexo server

Github准备

　　首先注册一个『GitHub』帐号，已有的默认默认请忽略，没有的在此注册。

　　建立与你用户名对应的仓库，仓库名必须为『your_user_name.github.com』，比如我的是fantasynew.github.io。

　　添加SSH公钥到『Account settings -> SSH Keys -> Add SSH Key』
　　前两步比较简单，第三步具体说下如何添加SSH-Key，详细教程在这里。

　　首先设置你的用户名密码：

git config --global user.email "jingjiewuying@163.com"
git config --global user.name "fantasynew"

　　生成密钥：

ssh-keygen -t rsa -C "jingjiewuying@163.com"

　　输入文件路径：

D:\hexo\blog>ssh-keygen -t rsa -C "jingjiewuying@163.com"
Generating public/private rsa key pair.
Enter file in which to save the key (//.ssh/id_rsa): D:\git\myssh\ssh
Enter passphrase (empty for no passphrase):
Enter same passphrase again:
Your identification has been saved in D:\git\myssh\ssh.
Your public key has been saved in D:\git\myssh\ssh.pub.
The key fingerprint is:
b0:0c:2e:67:33:ab:c1:50:10:40:0a:ba:c1:80:59:22 jingjiewuying@163.com

　　上述命令若执行成功，会在H:\git\myssh目录下生成两个文件id_rsa和id_rsa.pub，最后两步：

  • 用文本编辑器打开ssh.pub文件，拷贝其中的内容，将其添加到Add SSH Key
  • 将id_rsa和id_rsa.pub拷贝至你Git安装目录下的.ssh目录，如D:\PortableGit-1.8.4.ssh

部署到Github

　　编辑_config.yml(在Ｄ:\hexo下)。修改repository为你自己的github账号。

deploy:
type: github
repository: https://github.com/zippera/zippera.github.io.git
branch: master

　　执行下列指令完成到Github的部署。

hexo generate
hexo deploy

　　记住：每次修改本地文件后，需要hexo generate才能保存。每次使用命令时，都要在D:\hexo目录下。
　　Okay,我们的博客已经完全搭建起来了，在浏览器访问zippera.github.io就能看到你的成就了！

tips

hexo现在支持更加简单的命令格式了，比如：

hexo g == hexo generate
hexo d == hexo deploy
hexo s == hexo server
hexo n == hexo new
